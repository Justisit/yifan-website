       翼帆--河北工程大学师生自主研发产品展示网        成立于2015年，依托邯郸最高学府--河北工程大学，是一家专注于移动互联网业务的高新技术发布、展示网站，是由河北工程大学学生自主管理、维护、运营的非盈利网站。


       网站为近30000名教师、职工、学生服务，共有n多件展示产品，包括生活、教育、医疗、科技、网络、环境、经济、政治、文化等各个领域，网站与中国移动、中国联通、中国网通、中国电信有着丰富的合作经验，先后获得了由信息产业部颁发的“中华人民共和国增值电信业务经营许可证”、“中华人民共和国电信与信息服务业务经营许可证”和由信产部批准的全国四网合一短消息类服务商。


       网站有河北工程大学创拓web设计工作室制作，业务内容也从一开始的商务短信发展到集团短信、集团彩信、语音短信、营销短信、二维码、企业APP应用、微信营销、开发制作及应用服务！成为邯郸著名的移动商务整体解决服务品牌网！
